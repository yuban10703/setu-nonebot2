# nonebot的色图姬插件

本项目是[opq-osc/OPQ-SetuBot](https://github.com/opq-osc/OPQ-SetuBot)的色图插件移植的

可能有bug...

欢迎提issues...

#### 详细信息看[wiki](https://github.com/yuban10703/setu-nonebot2/wiki) (还没写...)



#### 支持的功能:

- 每个群的配置文件独立
- 可以控制频率
- 可以选择开关R18
- 3个API可以自定义开关
- bot的回复可以自定义
